<p style="font-size: 30px; font-weight: bold; text-align: center">Qii-UI</p>


一个轻量级、现代化，使用 TypeScript 编写的 Vue3 组件库。

- ⬆️ 支持按需引入
- ✨ 现代化设计
- 🌤️ 轻量级架构
- 🌈 支持自定义


### 安装
使用 npm 命令快速安装组件库。
```npm
npm i qii-ui
```
