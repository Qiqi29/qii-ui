<h1 align="center">Qii-UI</h1>


一个轻量级、现代化，使用 TypeScript 编写的 Vue3 组件库。

- ⬆️ 支持按需引入
- ✨ 现代化设计
- 🌈 支持自定义


### 开发中
组件库目前正在缓慢开发中，组件数量也不多，不推荐在正式项目中使用。


### 安装
使用 npm 命令快速安装组件库。
```npm
npm i qii-ui
```


### 文档
请查阅 [Qii-UI 组件库文档](https://qiqi29.github.io/qii-ui-doc/)。
